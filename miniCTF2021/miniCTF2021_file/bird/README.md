>Một game kiểu flappy bird khá là đơn giản, mục tiêu đã được nói rõ trong gợi ý là đạt 222 điểm.

-Với mục tiêu như thế này, thì hoàn toàn có thể chơi như bình thường để đạt mục tiêu.
